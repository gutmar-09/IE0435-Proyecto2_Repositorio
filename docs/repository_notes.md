# Notas del repositorio

Este repositorio fue organizado para cumplir con los entregables finales del proyecto:

- `README.md`: explica cómo instalar dependencias, entrenar modelos y ejecutar inferencia.
- `DATASET.md`: describe el conjunto de datos, su construcción, limitaciones y criterios de etiquetado.
- `LICENSE`: define los permisos de uso del código.
- `requirements.txt`: lista las dependencias para instalación con pip.
- `environment.yml`: permite crear un ambiente de conda.
- `reports/`: contiene el informe final del proyecto.
- `video/`: carpeta destinada al video demostrativo.
