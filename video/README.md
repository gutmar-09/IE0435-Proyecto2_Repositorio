# Video de validación

En esta carpeta debe colocarse el video demostrativo del funcionamiento completo del sistema.

El video debe mostrar, preferiblemente, las siguientes etapas:

1. Ejecución del nodo de reconocimiento de voz.
2. Pronunciación de la palabra de activación `Husky`.
3. Emisión de un comando de voz válido.
4. Reconocimiento del comando mediante Vosk.
5. Clasificación de la intención mediante el modelo entrenado.
6. Extracción del objetivo inicial y objetivo final.
7. Generación automática del archivo `operation_mode_parameters.txt`.
8. Verificación del modo de operación seleccionado.

Si el video es demasiado pesado para almacenarse directamente en GitHub o GitLab, se recomienda colocar aquí un enlace institucional al archivo.
