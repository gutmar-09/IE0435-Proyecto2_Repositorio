# Video de validación

En esta carpeta debe colocarse el video demostrativo del funcionamiento completo del sistema.

Como consideración importante, cada destacar que el video no se muestra el recorrido completo del robot dado que el tiempo de simulación era mucho.

Además, el desplazamiento del robot no es un aspecto que competa directamente en este proyecto, puesto que el proposito de este proyecto radica en el a creación del algoritmo de clasificación de comandos de voz y no en la ejecución de la trayectoria.
